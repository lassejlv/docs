## Installation

Install Prismo using your package manager of choice.

<ResponseField>
  <CodeGroup>
    ```json npm
    npm install prismo@latest
    ```
    ```json pnpm
    pnpm install prisma@latest
    ```
    ```json bun
    bun add prismo@latest
    ```
  </CodeGroup>

</ResponseField>

## Setting up Prismo

```ts
import { PrismoClient } from "prismo";

const db = new PrismoClient({
  url: "<database-url>",
  token: "<token>",
});
```

## Querying the database

Basic Example of querying the database:

```ts
const guild = await db.create<Guild>({
  table: "Guild",
  data: {
    id: crypto.randomUUID(),
    guildId: "1234567",
    prefix: "!",
    createdAt: new Date(),
    updatedAt: new Date(),
  },
});
```
